# Kanban Board UI

A Kanban board UI for task management with drag-and-drop columns. Organize tasks across To Do, In Progress, and Done stages.

## Features

- Drag and drop tasks between columns
- Add new tasks to any column
- Edit and delete existing tasks
- Visual status indicators
- Clean, modern UI

## Running it

```bash
git clone https://github.com/kunalshahdev/Kanban-Board-UI.git
```

Open `index.html` in any browser.

## Stack

HTML · CSS · Vanilla JavaScript
